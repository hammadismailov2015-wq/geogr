# 🌍 Гео-Тренажёр (6 класс)

Интерактивный веб-тренажёр по географии для 6 класса с уровнем «посложнее».
Работает полностью в браузере, без установки и интернета.

## Темы

| Тема | Что тренируем |
|------|---------------|
| 🌊 **Океаны** | Мировой океан, 5 океанов, глубины, течения, солёность, части океана |
| ⚓ **Моря** | Внутренние и окраинные моря, солёность, известные моря |
| 🏞️ **Реки** | Исток, устье, русло, приток, речная система, бассейн, водораздел |
| 🌤️ **Атмосфера** | Слои атмосферы, температура, давление, ветер, погода и климат |

## Возможности

- **Тренировка по темам** — выбери тему и отвечай на вопросы.
- **Экзамен** — 20 вопросов по всем темам вперемешку.
- **Карточки-факты** — сначала изучи теорию, потом проверь себя.
- Мгновенная проверка с **пояснением** к каждому ответу.
- Очки, серии правильных ответов (🔥) и личные рекорды по темам.
- Итоговый экран с процентом и советом, какие темы повторить.
- Тёмная и светлая темы, адаптивный дизайн (телефон / планшет / ПК).
- Управление с клавиатуры: клавиши `1`–`4` — ответ, `Enter` — «Дальше».

Прогресс сохраняется в браузере (localStorage).

## Как запустить

Просто открой файл **`index.html`** в браузере — всё готово.

Или запусти локальный сервер:

```bash
python3 -m http.server 8000
# затем открой http://localhost:8000
```

## Структура

```
index.html   — разметка и экраны
styles.css   — оформление, тёмная/светлая темы
data.js      — банк вопросов и фактов по темам
app.js       — логика квиза, навигация, сохранение прогресса
```

Чтобы добавить вопрос — открой `data.js` и допиши объект в массив `questions`
нужной темы: `{ q, options, answer, explain }`, где `answer` — индекс верного
варианта (с нуля).
